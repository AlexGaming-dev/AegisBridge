Is This Release Safe?

Before installing, verify you have the official release:

  • Download only from this page or the official GitHub Releases page.
  • Review the release notes for expected behavior and known limitations.
  • Verify the file checksum (SHA-256) against published values when available.
  • Prefer signed installers in production and managed environments.

SmartScreen / Smart App Control

Windows may display a SmartScreen or Smart App Control prompt for unsigned or new binaries. This is normal behavior. If you see a warning, verify that the installer came from the official releases page before proceeding.
