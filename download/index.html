Distribution

Installer packages and deployment channels.

Windows Desktop App

End users can download the installer directly. No source code checkout is required.

Download Windows Installer (.exe)

File: AegisBridge-OS-Guard-Setup.exe

Windows Installer Media Path

If your organization provides official installer media, store it in the following folder:

assets/windows-installer/11/installer.iso

The media file itself is not included in this repository.

Internal Build Command

This command is for maintainers, not for end users:

npm run build:win

CLI Validation Example

From the installer folder, run directory validation:

dir

Unsupported Command Workflows

Commands intended to bypass platform security or hardware requirements are intentionally excluded from this documentation.
